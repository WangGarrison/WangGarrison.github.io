Github与hexo搭建的个人博客网站更新后显示404

2020-05-17

更新博客后
hexo s 本地显示正常，但是hexo d提交线上后，用域名打开显示404

解决办法

hexo d后，登录Github找到自己博客的仓库，
点击settings

拉到底部将自己的域名填入，（虽然之前填过，但更新后有时要再填一次）

然后再点击rename，（虽然这没改动，但仍要点击一次）

操作完之后就可以用域名打开网站了
