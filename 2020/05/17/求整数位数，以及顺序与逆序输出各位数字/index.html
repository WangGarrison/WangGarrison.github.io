求整数位数，以及顺序与逆序输出各位数字

2020-05-17

题目背景：给一个正整数， 1.求它的位数，2.逆序输出每一位数字，3顺序输出每一位数字

1求位数，
算法: 每次丢弃个位数字,计数器++,直到n为0
与我们用眼睛去数类似，即不断给该整数除以十，计数，此处要考虑当给的整数位0时的特殊情况，C代码如下：

int Count(long long n) 
{
    if(n == 0)
        return 1;
    int tmp = 0;
    while(n != 0)
    {
        tmp++;
        n /= 10;
    }
    return tmp;
}

2.逆序输出
算法：求出低位，输出低位，再丢弃低位，重复执行直到n为0
这里关键是求低位的算法，用除十取余来得到低位 ，C代码如下：

void ReverseOutput(int n)
{
    if(n == 0)
    {
        printf("0\n");
        return;//提前结束该函数
    }
    else if(n < 0)
    {
        printf("-");
        n = -n;
    }

    while(n != 0)
    {
        printf("%d ",n%10);
        n /= 10;
    }
    printf("\n");
}

3.顺序输出
算法:得到高位,输出高位,丢弃高位,直到n为0，
这里关键是得到高位的算法，C代码如下：

void OrderOutput(int n)
{
    int tmp = Count(n); //得到n是几位数字
    int power = (int)pow(10.0, tmp - 1);//获得高位
    for (int i = 0; i < tmp; i++)
    {
        printf("%d ", n / power);//输出最高位
        n %= power;//丢弃最高位
        power /= 10;
    }
}
